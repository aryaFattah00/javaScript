    Tipe Data array

  • array adalah tipe data yang berisikan kkumpulan data
  • array di javaScript memiliki sifat dinamis, artinya datanyabisa bertambah dengan sendiri saat kita memasukkan data ke dalam array

operasi di Array

Operasi Keterangan
array.push(value) menambah data ke array
array.length untuk mendapatkan panjangg Array
array[index] mendapat data di posisi panjang array
array[index] = value mengubah data di posisi index
delete array[index] = value menghapus data di posisi index, namun index tidak bergeser

perlu di ingat jika kita bisa memasukkan Array ke dalam Array jika kita mau/ disebut multi dimensional array
