Falsy

  • falsy atau kadang ditulis fslsey, adalh value yang ketika dalam konteks boolean, dia dianggap false
  • ini dianggap salah satu fitur unik javaScript, yang berguna,namun kadang juga seriing membingungkan
  • di javaScript, kondisi ini hanya bisa boolean, tapi diluar boolean pun bisa, namun kita harus tau beberapa data falsy, atau dianggap false

Data Falsy

Data dianggap Falsy Keterangan
False Boolean false
0, -0 Number 0 dan -0 dianggap false
"", '', `` Semua angka kosong dianggap false
null null dianggap false
undefiend undefiend dianggap false
NaN Not a Number dianggap false

    Truthy

  • Truthy adalah kebalikan dari falsy, dimana data nya dianggap bernilai true
  • sederhana sekali sebenarnya untuk mengetahui senuah data itu adalah truthy, yaitu data yang bukan bernilai falsy
