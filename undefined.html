Undefiend Variable

  • sebuah variable yanng belum ditambahkan nilai, maka artinya variable tersebut merupakan tipe undefiend
  • Kadang untuk programmer javaScript pemula undefiend ini memang agak sedikit membingungkan
  • undefiend itu berbeda dengan null di bahasa pemrograman lainnya
